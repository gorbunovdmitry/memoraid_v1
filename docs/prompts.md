# Черновики промптов (Gemini)

## Классификация запроса
Инструкции:
- Определи класс: `memory | calendar | advice | audio`.
- Для calendar: извлеки дату/время/таймзону (если нет — пометь как requires_clarification).
- Для memory: предложи папку из списка: Работа, Учёба, Семья, Друзья, Хобби, Места, Здоровье, Спорт, Дом.
- Для advice: верни краткую формулировку запроса.
- Для audio: если это запрос про аудио/диктофон.

Формат ответа (JSON):
```
{
  "kind": "memory|calendar|advice|audio",
  "folder": "Работа|Учёба|Семья|Друзья|Хобби|Места|Здоровье|Спорт|Дом|null",
  "datetime": "ISO8601|null",
  "needs_clarification": false|true,
  "notes": "строка"
}
```

## Память (нормализация факта)
Цель: лаконично сохранить факт.
Инструкции:
- Верни `title` (очень коротко) и `content` (1–3 предложения, без воды).
- Не выдумывай деталей; пиши на русском.
- Если папка не выбрана — предложи наиболее подходящую из списка.
Формат ответа:
```
{ "title": "...", "content": "...", "folder": "..." }
```

## Календарь
Инструкции:
- Извлеки цель события, дату/время (используй LLM + chrono-ru в коде; в ответе верни нормализованное значение).
- Если дата/время неполные — `needs_clarification: true`.
Формат:
```
{
  "title": "...",
  "description": "...",
  "starts_at": "ISO8601|null",
  "ends_at": "ISO8601|null",
  "needs_clarification": false|true
}
```

## Совет
Инструкции:
- Учитывай прошлые факты/предпочтения (контекст передаётся в prompt).
- Отвечай конкретно, 3–5 предложений, без общих фраз.

## Диктофон (транскрипция → summary)
Инструкции:
- Вход: транскрипт аудио.
- Верни: `summary` (3–6 предложений) и `next_steps` (bullets, лаконично).
- Пиши на русском.
Формат:
```
{
  "summary": "...",
  "next_steps": ["...","..."]
}
```

## Тон и стиль
- Русский язык, вежливо и кратко.
- Не добавлять маркетинга/эмодзи.
